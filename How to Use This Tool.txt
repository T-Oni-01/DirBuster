How to Use This Tool (For Refresher):

* Install required dependencies: "pip install requests"

* If you don't have a Python IDE, run in terminal with: "python dirbuster.py http://example.com" (please me, remember to navigate to the file location first).

* Customizations:

# Basic scan
python dirbuster.py http://example.com

# Scan with custom wordlist and 20 threads
python dirbuster.py http://example.com -w custom_wordlist.txt -t 20

# Scan with specific timeout
python dirbuster.py http://example.com --timeout 3
